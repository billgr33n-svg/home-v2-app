Home v2 — interactive demo

A faithful reproduction of the app's Today, Rides, Meals, News, and Polls tabs, running the real domain logic against the Green-household seed data. Every tap works. This is the UI and logic, not the native build — real push and the live database attach when the Expo app boots on a device.

Try this

  • Rides → Claim driver. Return to Today: the ride drops from a red "needs a driver" decision to a quiet blue line.
  • Meals → answer dinner. You're the outstanding response. Tap Home and watch Today's dinner flag clear.
  • Polls. Create a poll, answer it, close it.
  • News. Post an announcement.
  • Shop. Add a shopping item, check it off, and tap an approximate pantry item to cycle its level.
  • Tasks. Claim the Unassigned task, or complete one. Recurring chores are badged.
  • Upkeep. The furnace filter is overdue — mark it done to reset the cadence.

What you won't see

  • The Rivera household's data — every query is scoped to your household. That boundary is enforced in the database (row-level security), proven separately; here it's just reproduced.

Green household seed: Bill, Sandy, Will, Matt, Cora. Demo is single-user (you act as Bill).
